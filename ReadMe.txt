The leading bank aims to predict credit card consumption for customers based on demographic and behavioral data, focusing on predicting the average spend for the next three months. Key objectives include:

Conducting data exploratory analysis on customer demographics and behavior.
Building predictive models to estimate credit card consumption for customers with missing values.
Validating models using Root Mean Square Percentage Error (RMSPE).

Expected Outputs:
Detailed code with comments for data analysis and model building.
Exploratory analysis results highlighting insights from customer data.
Model validation outputs including RMSPE.
Comprehensive model documentation with details on methodology and results.
Predicted values for customers with missing credit card consumption data.
